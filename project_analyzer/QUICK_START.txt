📌 ШАГ 1: УСТАНОВКА ЗАВИСИМОСТЕЙ

   Откройте командную строку в этой папке и выполните:

   pip install -r requirements.txt


📌 ШАГ 2: НАСТРОЙКА API КЛЮЧА

   1. Создайте файл .env (если его нет)
   2. Добавьте в него свой OpenRouter API ключ:

      OPENROUTER_API_KEY=sk-or-v1-ваш_ключ_здесь

   Пример в файле: .env.example


📌 ШАГ 3: ЗАПУСК

   ⚡ ПРОСТО ЗАПУСТИТЕ: start.bat

   Сервер автоматически:
   ✓ Поднимется на http://localhost:8000
   ✓ Откроет браузер
   ✓ Будет готов к работе!


📌 ОСТАНОВКА СЕРВЕРА

   Запустите: stop.bat

   Или нажмите Ctrl+C в окне сервера


╔═══════════════════════════════════════════════════════════════╗
║                      БЫСТРОЕ ИСПОЛЬЗОВАНИЕ                    ║
╚═══════════════════════════════════════════════════════════════╝

1. Введите ПОЛНЫЙ путь к проекту в поле (например: E:\my_project)
2. Нажмите "Start Analysis" или Enter
3. Дождитесь завершения (2-3 минуты)
4. Изучайте граф зависимостей и проблемы!

⚠️ ВАЖНО: Указывайте АБСОЛЮТНЫЙ путь к папке проекта!
   Правильно:   E:\project_analyzer\test_project
   Правильно:   C:\Users\Name\Documents\my_code
   Неправильно: test_project (относительный путь)
   Неправильно: .\my_folder (относительный путь)

❌ Проблемы с зависимостями?
   → Переустановите: pip install -r requirements.txt --force-reinstall

❌ Порт 8000 занят?
   → Измените UI_PORT в config.py

❌ Ошибка API ключа?
   → Проверьте .env файл и корректность ключа
