╔═══════════════════════════════════════════════════════════════╗
║                      ЧТО ДЕЛАТЬ ЕСЛИ...                       ║
╚═══════════════════════════════════════════════════════════════╝

❌ "Python not found"
   → Установите Python 3.8+ с https://python.org
   → При установке отметьте "Add Python to PATH"

❌ "Port 8000 is already in use"
   → Запустите stop.bat
   → Или измените порт в config.py

❌ "ModuleNotFoundError"
   → Выполните: pip install -r requirements.txt

❌ "OpenRouter API error"
   → Проверьте файл .env
   → Добавьте корректный OPENROUTER_API_KEY

❌ Сервер запустился но браузер не открылся
   → Откройте вручную: http://localhost:8000

❌ Окно сервера мелькнуло и закрылось
   → Запустите start.bat (он покажет ошибку)
   → Или запустите вручную: python ui/app.py
